# Churn-Music-Dashboard

🚀 🎧 🎵📊  Very excited to share my latest Power BI project — a Churn Analysis Dashboard built using a music‑streaming platform dataset!  

This dashboard focuses on understanding user behavior, subscription patterns and early churn indicators to help identify at‑risk users and uncover opportunities to improve retention. Turning raw usage data into meaningful insights allowed me to explore how data‑driven storytelling can support smarter decision‑making in subscription‑based businesses.

To build this solution, I worked through the complete Power BI workflow — from data cleaning, modeling to creating DAX measures and designing interactive visuals. I developed key metrics such as churn rate, retention rate, engagement segmentation, and subscription plan performance. The final dashboard includes intuitive KPIs, trend lines, segmentation views and drill‑down filters that help stakeholders explore churn patterns across different user groups with ease.
This project strengthened my skills in data modeling, DAX Calculations, dashboard design and business‑focused analytics. More importantly, it helped me understand how to translate complex datasets into actionable insights that support user retention strategies. 

📌 Total users, Churned users, Active users and Churn rate KPI’s

📌 Gender-wise and Age-group-wise churn analysis

📌 Churn Rate by device Type, Network type, Top genre and Buffering Time

📌 Subscription plan churn comparison on Current plan, previous plan 

📌 DAX measures for Current plan, previous plan, churn difference and churn % change

📌 Slicer using payment method and auto-renewal

🔧 Tools & Skills Used:
The dashboard was built using the following tools and technologies:

• 📊 Power BI Desktop – Main data visualization platform used for report creation.

• 📂 Power Query – Data transformation and cleaning layer for reshaping and preparing the data.

• 🧠 DAX (Data Analysis Expressions) – Used for calculated measures, dynamic visuals, and conditional logic.

• 📝 Data Modeling – Relationships established among tables (resorts, snow, and data_dictionary) to enable cross-filtering and aggregation.

• 📁 File Format – .pbix for development and .png for dashboard previews.

• 📘 End-to-end analytical storytelling


 🚀Business Impact & Insights Marketing Optimization 🚀:


• Overall churn rate is 31%, meaning nearly 1 in 3 users leave the platform.

• Out of 4,600 total users, 1,412 churned and 3,188 remain active.

• Male users churn 4 percentage points higher than female users (71% vs 67%), indicating slightly weaker retention among men.

• Youngest users (18–25) show the highest churn rate at 32%, which is 2% higher than older age groups.

• Churn is consistent across devices, with only a 1% difference between Android, iOS, and Web — no device‑specific retention issue.

• 5G users churn 2% more than WiFi/4G users, possibly due to higher expectations for speed and performance.

• Free plan churn increased the most, with a +1.48% absolute rise and +5.41% relative increase.

• Family plan churn increased slightly, with a +0.71% absolute rise and +2.22% relative increase.

• Premium plan churn improved significantly, showing a –2.07% absolute drop and a –6.33% relative improvement, indicating stronger loyalty among paying users.
 
 This dashboard helps to make business decisions based on actual data not any guess work.
 Churn remains at 31%, with the highest risk among younger users and male users. Device and network performance show minimal impact, indicating churn is driven more by user experience and content preferences than technical issues. Premium subscribers show a strong 6.33% improvement in retention, while Free users show the largest deterioration at 5.41%. These insights highlight the need for targeted retention strategies for younger demographics and free‑tier users

 The leadership team can use these insights to take actions on ways to reduce the churn such as 
 
 ✅Enable Auto-Renewal by Default
 
 ✅ For Free-plan subscription users, Trial extensions and limited time discounts
 
 ✅ Premium content teasers can be introduced to Family and Free subscription users
 
 ✅ Subscription Renewal reminders can set to users
